<!-- GETTING STARTED -->
## Getting Started
1. Install Angular
2. Clone the repo:
```
git clone https://github.com/salvasm/angular-auth.git
```

3. Install NPM packages:
```
npm install
```

4. Build and run server command:
```
npm run build && npm run start
```
